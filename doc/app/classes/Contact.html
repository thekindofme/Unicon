sync_safe_delete()

delete a contact in ‘sync safe way’ if the contact is a already synced one, it will not be deleted right away. its deleted will be set to true so that when it‘s contact source is synced, its delete will be synced with the remote contact sources, and then it will be deleted from unicon‘s systems if the contact is not synced already it will be just deleted
